# Environments and API Keys

You can add as many environments as you need. Each environment comes with an API Key that be can used for with our clients.

One Environment can be set as "Production" - this is just for your record keeping.

## API Key

You can get a copy of your API key, by going to the Environments section, clicking on _API KEY_ for an environment, and clicking the "Copy" button

You can reset an API key, by going to the Environments section, clicking on _API KEY_ for an environment, and clicking _Generate API key_
